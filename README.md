# 🤖 Viral YouTube Shorts Automation Agent

A fully autonomous AI pipeline that generates, edits, and uploads viral YouTube Shorts 24/7. Built with Python, Groq AI, and FFmpeg.

## 🚀 Features
- **AI Scripting:** Uses Groq (Llama 3) to write highly engaging, viral scripts.
- **Neural Voiceover:** Generates hyper-realistic human voices using Microsoft Edge-TTS (100% free).
- **Motion Backgrounds:** Automatically pulls high-quality, vertical stock videos from the Pexels API.
- **Hormozi-Style Captions:** Uses FFmpeg to burn bold, animated, center-screen subtitles directly onto the video.
- **Auto-Publishing:** Seamlessly uploads and publishes videos directly to YouTube via the YouTube Data API.

## 🛠️ Tech Stack
- Python 3.14
- Groq API (AI Generation)
- Pexels API (Video Assets)
- Edge-TTS (Audio)
- FFmpeg (Video Processing)
- Google YouTube Data API v3 (Automation)

## ️ How to Run
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Add your API keys to a `.env` file.
4. Run the agent: `python final_viral_system.py`
